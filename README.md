# CPP MODULES

This repository contains a collection of C++ exercises and projects covering the core concepts taught in the CPP Modules series.

## Overview

The project is organized by module folders:

- cpp00: Basics of C++ and introductory exercises
- cpp01: Memory allocation, references, and pointers
- cpp02: Orthodox Canonical Form and fixed-point numbers
- cpp03: Inheritance and polymorphism basics
- cpp04: Abstract classes and subtype polymorphism
- cpp05: Exceptions and forms
- cpp06: Type casting and conversion
- cpp07: Templates
- cpp08: Templated containers and algorithms

## Project Structure

Each exercise folder contains its own source files, header files, and a Makefile. To compile and run a specific exercise, navigate to its directory and run:

```bash
make
./<program_name>
```

## Requirements

- A C++ compiler supporting C++98 (or later)
- GNU Make

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/abdoihlane/CPP-MODULES.git
   ```
2. Enter the project folder:
   ```bash
   cd CPP-MODULES
   ```
3. Compile an exercise:
   ```bash
   cd cpp00/ex00
   make
   ```
4. Run the executable:
   ```bash
   ./megaphone
   ```

## Notes

This repository is intended for learning and practicing C++ concepts through small, structured exercises.
